Questions & Negation

Ke Kero Keti — What, Where, When
Vaelic forms all questions using a single particle system. Yes/no questions begin with Ke. Information questions use dedicated question words (Kena, Kero, Keti, Keno, Nekati, Kenozari). Negation is achieved by prefixing No- directly to the verb. Both systems are completely regular — no irregularities, no special verb forms, no question inversion.
Learning Objectives
  • Form any yes/no question using the Ke particle
  • Use all seven question words correctly in context
  • Negate any verb using the No- prefix
  • Use the urgency particle Ra appropriately
  • Combine negation with questions in complex interrogatives
