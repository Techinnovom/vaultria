21.2 Canonical Compound Patterns

21.2

Canonical Compound Patterns

Compound Pattern Types

[ACTION] + [OBJECT] = result action: Birek + Kat = Birekkat (demolish = build + stop)

[NOUN] + [NOUN] = composite place/thing: Nod + Dekat = Noddekat (city center)

[NOUN] + [DESCRIPTOR] = qualified noun: Zelek + Morat = Zelekmorat (stronghold = strong place)

[ACTION] + [ACTION] = combined action: Kavi + Terina = Kaviterina (explain = speak + teach)

[ELEMENT] + [DANGER] = warning compound: Varek + Drak = Varekdrak (flood = water + danger)
